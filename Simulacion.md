# Mini-Proyecto 1: Sistema de Gestión de Tareas Académicas

## Integrante
- Luis Emilio Arias Santizo

## Definición del problema
Los estudiantes suelen tener dificultades para organizar sus tareas académicas, lo que provoca retrasos, olvidos y mala gestión del tiempo.
Actualmente, no todos utilizan herramientas adecuadas para priorizar actividades o visualizar su progreso, lo cual afecta su rendimiento académico.

## Funcionalidades principales
El sistema propuesto contará con las siguientes funcionalidades:
1. Crear tareas académicas
2. Asignar prioridad (alta, media, baja)
3. Marcar tareas como completadas
4. Visualizar lista de tareas pendientes
5. Filtrar tareas por estado o prioridad

## Roles del sistema
- **Estudiante:** Usuario principal que crea y gestiona sus tareas
- **Administrador:** Puede supervisar o gestionar usuarios

## Flujo de uso
1. El usuario inicia sesión
2. Crea una nueva tarea
3. Asigna prioridad
4. Visualiza tareas pendientes
5. Marca tareas como completadas
6. El sistema actualiza el progreso
